# 🎓 Certificate Generator System

A full stack certificate generation system built for Amaanitvam Foundation.

## Features

- Generate certificates
- Download PDF certificates
- QR Code support
- Certificate verification
- MongoDB database integration
- Admin dashboard
- Search certificates

## Tech Stack

Frontend:
- React.js
- Vite

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

## Installation

Frontend:

npm install
npm run dev

Backend:

cd server
npm install
node server.js
